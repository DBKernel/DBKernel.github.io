1.11. 补充：如何将所有节点部署在一台主机？

如果要将所有节点配置在一台主机，比如：在 h93 配置2个主 segment、2个镜像 segment、1个 master，只需要把hostfile_config中的 h94 删掉，然后在 h93 删除 data/primary，data/mirror，data/master 目录下的内容，重新初始化即可。
