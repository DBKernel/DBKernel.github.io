## 使用说明

1. 添加文章：在 当前路径/content/post/ 中创建目录和md文件，比如是 mysql-test/MySQL测试用例详解.md。
2. 执行`hugo server -D`在终端启动服务。
3. 在浏览器输入 `http://localhost:1313/` 预览。
4. 若文章无问题，在当前路径执行`hugo`构建静态网站目录`public`。
5. 将 `public` 路径下的文件挪到 `../` 即上级目录。
6. 提交到github。
